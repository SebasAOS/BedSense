Características Principales

Bedsense ofrece una solución completa para la gestión de recursos hospitalarios

Monitoreo en Tiempo Real

Seguimiento instantáneo de la disponibilidad de camas en todas las áreas del hospital, con actualizaciones automáticas y sin demoras.

Gestión Eficiente

Visualización clara de camas ocupadas y disponibles, optimizando la asignación de recursos y reduciendo tiempos de espera.

Base de Datos Segura

Utilización de Google SQL con MySQL para garantizar un almacenamiento confiable, escalable y seguro de los datos hospitalarios.

Reportes y Análisis

Generación de informes detallados y análisis de tendencias para la mejora continua de los procesos hospitalarios.

Acceso Multiplataforma

Interfaz responsive que funciona en computadoras de escritorio, tablets y dispositivos móviles para acceso en cualquier lugar.

Alertas Automáticas

Notificaciones en tiempo real cuando se liberan camas o cuando se alcanzan niveles críticos de ocupación.

Instalación

Para instalar y configurar Bedsense en tu entorno hospitalario, sigue estos pasos:

  • Clona el repositorio de GitHub
  • Configura el entorno virtual de Python
  • Instala las dependencias con pip
  • Configura la conexión a Google Cloud SQL
  • Ejecuta las migraciones de base de datos
  • Inicia el servidor de desarrollo

Las instrucciones detalladas estarán disponibles una vez el proyecto esté completo.
